# 数学知识点模块

根据数学知识设计清单创建的模块目录结构。

## 目录结构

```
modules/
├── grade-1/
│   ├── number-recognition/ (数字认知)
│   ├── addition-subtraction-within-10/ (10以内加减法)
│   └── simple-shapes/ (简单图形)
├── grade-2/
│   ├── addition-subtraction-within-100/ (100以内加减法)
│   ├── multiplication-tables/ (乘法口诀)
│   └── length-measurement/ (长度测量)
├── grade-3/
│   ├── perimeter-and-area/ (周长与面积)
│   ├── fraction-basics/ (分数基础)
│   └── time-concept/ (时间概念)
├── grade-4/
│   ├── decimals/ (小数)
│   ├── factors-and-multiples/ (因数与倍数)
│   └── angles/ (角度)
├── grade-5/
│   ├── fraction-operations/ (分数运算)
│   ├── volume/ (体积)
│   └── coordinate-system/ (坐标系)
└── grade-6/
    ├── negative-numbers/ (负数)
    ├── ratios-and-proportions/ (比例)
    └── simple-equations/ (简易方程)
```
